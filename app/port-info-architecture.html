Work Flow

During the initial phase of a site redesign, two user groups were identified. Each had a slightly different flow when visiting the site. The workflow diagram guided the process.

Site Map

The original site was complex and the navigation did not support way-finding. The new organizational structure is simpler and brings vital information about the company's offerings to the main pages.

Wireframes

I created wireframes of the new pages using Axure. This particular page redesign addressed 5 main issues:

  1. Users could not view all of the data they wished to see
  2. Travel packages were not discoverable
  3. Users wished to compare options
  4. The ability to "Learn More" was added for confused users
  5. Users wanted to be able to contact support via email
